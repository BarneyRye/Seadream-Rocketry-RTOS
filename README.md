# Seadream Rocketry PCB and Code
 Kicad files include schematic, pcb, extra symbols, footprints 3d files and also the production files for JLCPCB </br>
 Included code written in Platform.io on VSCode, uses prebuilt libaries and freeRTOS, one task for sensor reading and audio, other task for writing to SD </br>
 Seperate repository "https://github.com/BarneyRye/Seadream-Rocketry-Avionic-Log-Breakdown.git" has code to breakdown log file and analayse the data to plot trajectory
